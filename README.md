# 🏆 Copa do Mundo 2026 API

Uma API RESTful desenvolvida em Python utilizando o framework **FastAPI** para fornecer informações completas sobre os grupos, seleções e chaves oficiais da Copa do Mundo FIFA 2026.

## 🚀 Tecnologias Utilizadas

*   **Python 3.10+**
*   **FastAPI:** Framework moderno, rápido (alta performance) e fácil de codificar.
*   **Pydantic:** Validação de dados e definição de Schemas.
*   **Uvicorn:** Servidor ASGI de alta performance para rodar a aplicação.

---

## 📌 Funcionalidades

*   Listagem completa de todos os 12 grupos (A ao L) e suas respectivas 48 seleções.
*   Busca detalhada de um grupo específico através da sua letra identificadora.
*   Listagem global de times participantes.
*   Filtro dinâmico de seleções por continente (Ex: `CONMEBOL`, `UEFA`, `CAF`).
*   Documentação interativa automática integrada (Swagger UI).
